# form-builder

 #### 技术栈： 基于VUE2.x + ant-design-vue 1.6.2 + sortable.js(拖拽) 实现的动态表单功能，支持编辑模式，表格模式，详情模式


 #### 代码按单元组件模块式划分，结构清晰、维护简单

[演示地址：https://form.jaymw.site/](https://form.jaymw.site/)
